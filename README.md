# 🎧 Tokyo Shadowing Master

一個可直接在瀏覽器使用的 **日文 Shadowing 練習工具**
整合「東京一日遊導覽」＋「餐廳對話實戰」，適合初級～中級學習者。

---

## 🌟 功能特色

### 🗼 一日遊導覽模式

* 旁白式日文（自然語感）
* 涵蓋東京經典動線：

  * 淺草 → 上野 → 秋葉原 → 銀座 → 東京塔 → 晚餐
* 適合：

  * 聽力訓練
  * 整句語感建立

---

### 🍽️ 餐廳對話模式

* 客人 vs 店員雙角色
* 真實情境（日式餐廳）
* 支援：

  * 只聽客人（練開口）
  * 只聽店員（練聽力）

---

### 🎯 Shadowing 強化功能

* ▶ 全部播放 / 單句播放
* ⏮ ⏭ 上一句 / 下一句
* ⏸ 暫停 / ⏹ 停止
* 🎚 語速調整（預設 0.80x）
* ⏱ 每句停頓（預設 1000ms）
* 🔁 重播次數設定

---

### 🎨 高級 UI 設計

* iPhone 風格 **Segmented Control（滑動切換）**
* 當前句「淡藍高亮」
* 滑塊動畫 + 陰影（高級質感）

---

## 🚀 使用方式

### 方式一（本機）

1. 下載 `index.html`
2. 直接雙擊打開（Chrome / Edge）

---

### 方式二（GitHub Pages）

1. 建立 repository
2. 上傳 `index.html`
3. 到 Settings → Pages
4. 選擇：

   * Branch: `main`
   * Folder: `/root`
5. 開啟網址：

```text
https://你的帳號.github.io/專案名稱/
```
